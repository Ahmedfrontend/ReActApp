1 - unrar file RizkallaApp

2- open CMD (windows command line ) as administrator and go to RizkallaApp folder


3- make sure you install node.js by run this command node -v if you not have node.js please install it from 
https://nodejs.org/en/


4- npm install

5 - npm start


if you get any error when run or install please install webpack-cli global by run this command

npm install -g webpack-cli
